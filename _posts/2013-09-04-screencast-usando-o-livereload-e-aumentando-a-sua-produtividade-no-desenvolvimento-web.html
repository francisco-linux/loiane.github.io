Ei pessoal,

Hoje vim compartilhar uma dica rápida de ferramenta para melhorar a produtividade de quem trabalha com desenvolvimento web. Esse ferramenta se chama LiveReload.

O seu uso é bem simples: basta adicionar um script na página HTML (temporariamente, apenas enquanto desenvolve, depois você remove quando for para produção) e quando você salva algum arquivo, o browser automaticamente faz refresh. Pode parecer besteira, mas isso salva um bom tempo do nosso dia a dia.

Para Mac OS é pago e custa 9.99 obamas. Para Windows e Linux é free.
