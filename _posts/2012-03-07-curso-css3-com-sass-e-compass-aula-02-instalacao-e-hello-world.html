Vídeo/Screencast de hoje: aula 02 do curso gratuito de CSS3 com Sass e Compass!

Na aula de hoje vamos aprender como instalar o Sass, que é uma gem do Ruby - iremos ver como fazer isso em cada Sistema Operacional: Windows, Linux (Ubuntu) e Mac OS. Além disso, vamos aprender como fazer um hello world, ou seja, compilar um arquivo Sass em CSS para verificar a instalação.

Quem quiser ver todas as aulas já publicadas, é só clicar aqui.

Espero que gostem dessa aula e fico no aguardo do feedback de vocês! :)

Requerimentos:

  • Ruby
  • Gem Sass
  • Editor de Texto. Nas aulas estou usando o Sublime Text 2.
  • Terminal comum (Linux e Mac) ou Command Prompt do Ruby (Windows)
