Na aula de hoje vamos aprender como usar os comandos break e continue! O comando break é mais utilizado para sair de loops, enquanto o continue é um complemento do break, onde diz ao programa para continuar o loop na próxima iteração. Ambos podem ser usados com labels (rótulos), que seria um goto, mas é raramente utilizado no Java.
