O objetivo deste post é mostrar como importar um projeto pronto J2EE para o Eclipse e executá-lo após a importação

Para a demonstração, vou utilizar o projeto do post sobre 'Começando com Ext JS'.

Antes de começar, verifique se a versão do seu eclipse tem suporte para projetos J2EE.
Você pode verificar isso através do Help -> About Eclipse Software Platform (procure nas Feature Details por 'J2EE').
Outra forma é verificar se o Eclipse tem a perspectiva J2EE disponível:

Se você não tiver essa opção disponíve, faça o download em: http://www.eclipse.org/downloads/
Escolha a opção 'Eclipse IDE for Java EE Developers'

Vamos então começar!

O primeiro passo é colocar a pasta do projeto em algum diretório. Para facilitar, vou deixar dentro da pasta do meu worskpace mesmo:

Depois, no menu da IDE, escolha File -> Import

Abra a aba 'General' e escolha 'Existing Projects into Workspace' -> Next

Escolha o diretório onde está a pasta do projeto. Repare que a IDE reconhecerá o projeto:
Obs.: se você colocar o projeto no desktop, por exemplo, o projeto continuará lá, o Eclipse apenas fara a referência, ou seja, o seu projeto não estará dentro do diretório do workspace. Se você desejar que suas alterações fiquem dentro do workspace, marque a opção 'copy projects into workspace', assim, a IDE criará uma cópia do projeto na pasta do workspace. Como descompactei o arquivo dentro do wokspace mesmo, não marcarei esta opção.
Depois, clique em Finish.

Prontinho! O projeto vai aparecer no Project Explorer:

Pode ser que o projeto mostre alguns erros, seja pela versão do java compiler, seja porque não tem referência para um server (estou usando o Tomcat 5.5). Caso apareça algum erro, clique com o botão direito em cima do projeto e escolha Properties (veja a opção), ou Build Path -> Configure Build Path, e acerte os erros.

Agora o projeto está pronto para ser executado! :)

Para executar o projeto, siga os passos abaixo:

Na aba Servers, clique com o botão direito do mouse, escolha New -> Server:

Escolha o server. Neste caso, vou utilizar o Tomcat 5.5 que tenho instalado na máquina:

Se for a primeira vez que esteja configurando o server, a tela abaixo vai aparecer. Escolha o diretório onde você instalou o server.

Após isso, irá aparecer uma tela para escolher quais projetos você deseja adicionar ao server. Escolha o projeto que você importou e clique em Add:

Mas se você já tem um server configurado, na aba Server, escolha o server, clique com o botão direito e escolha a opção Add and Remove Projects...

Escolha o projeto que importou e clique em Add.

Agora basta iniciar o server e rodar a aplicação!
