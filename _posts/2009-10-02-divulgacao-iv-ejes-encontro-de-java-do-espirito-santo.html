Programação e resumo das palestras

09:20h - Aprendendo sobre NoSQL na prática com Grails e CouchDB

Quebrando paradígmas com uma abordagem prática a banco de dados não relacionais e como utilizar o CouchDB no Grails.

por André Lima Silva - Grupo Otto Andrade

André Lima é gerente de TI da Eximbiz Comércio Internacional S/A e consultor de tecnologia do Grupo Otto Andrade, cursando o 8 período de Sistemas de informação pela FAESA, entusiasta de tecnologias emergentes e metodologia ágil. Também é um dos coordenadores do ESJUG.

10:00h - Turbinando sua aplicação com Lucene

Pesquisa eficiente não deve ser um luxo apenas do Google, suas aplicações também precisam ser capazes de desempenhar pesquisas de modo eficiente e trazer resultados relevantes para o seu usuário. Para isso veremos um pouco sobre o Lucene, um projeto que auxilia na indexação, pesquisa e análise de textos e documentos. E veremos como podemos aproveitar essa excelente ferramenta utilizando o Solr: um servidor de pesquisas de alta performance implementado sob o Lucene e; Hibernate Search: como integrar o poder das pesquisas com Lucene diretamente em seu modelo de negócios.

por Paulo César M. Jeveaux - Giran Soluções e Ensino

Jeveaux é fundador e diretor na Giran Soluções e Ensino, onde trabalha com desenvolvimento de plataformas e serviços para internet de alta performance e escalabilidade utilizando metodologias ágeis como XP e Scrum. É fundador do Grupo de Usuários Java do Espírito Santo (ESJug), do grupo de metodologias ágeis (Agile-ES) e também administrador do PortalJava.com. Possui mais de 9 anos de experiência em desenvolvimento de sistemas em diversos segmentos, passando por globo.com, Vale, Claro, Xerox, CEF e outras. Grande entusiasta e participante ativo das comunidades de Java, Agile, Ruby e Linux.

10:50h - Keynote patrocinador: A definir

11:10h - JavaEE? Para onde você está indo? Conheça, compare e saiba escolher o melhor framework web

Desenvolver em java é vantajoso porque além de ser open source, nós temos diversos frameworks a disposição para as nossas aplicações. Contudo, como escolher um framework? Que tipo de decisões os desenvolvedores e arquitetos precisam tomar para escolher um framework? O framework é open source? Suporta Ajax? Tem testes? É action-like ou component-based? E a comunidade, é ativa? Nessa palestra serão mostrados os principais frameworks do mercado, discutiremos alguns deles e veremos via códigos as facilidades do framework brasileiro VRaptor.

por Rafael Carneiro - CEJUG

Rafael Carneiro trabalha com Java há 4 anos e meio com foco em JEE. Atualmente trabalha na TBM Têxtil como Analista de sistemas desenvolvendo aplicações JEE e é instrutor dos cursos de Formação Java da FA7. Atua ainda como moderador do GUJ, PortalJava e JUG Leader do CEJUG. Possui algumas certificações da Sun Microsystems e participa de projetos open source como o GUJ 3 e JForum. Mantém um blog sobre desenvolvimento de software no endereço rafaelcarneiro.com

12:30h - Parada para o almoço

14:00h - Desenvolvimento de aplicativos móveis com Java ME

Os dispositivos móveis vem ocupando um lugar de importância no mercado, levando informações estratégicas para fora das empresas sem que sejam necessários muitos aparatos tecnológicos, mas apenas um celular. E o que muda na hora de desenvolver para dispositivos móvies? O intuito desta palestra é mostrar quais as diferênças do Java ME e o que é possível fazer com Java ME para que o desenvolvedor comece a imergir nesta tecnologia.

por Bayron Thiengo Quinelato

Bayron Thiengo Quinelato é analista sistemas na Highlan Sistemas Inteligêntes onde atua no desenvolvendo soluções móveis na plataforma Java ME compatíveis com os sistemas Symbian, BlackBerry OS, Windows Mobile e outros sistemas proprietários. Cusando o último período de Ciência da Computação pela FAESA, é entusiasta da computação móvel e vem atuando na área desde 2004.

14:40h - Ajax de primeira com ExtJS + JSON no seu projetoSpring

O grid, é sem sombra de dúvidas, um dos componentes mais utilizados quando se trata de aplicações web, pois facilita tanto a vida do desenvolvedor quando a vida do usuário, devido sua simplicidade. E nesta palestra, veremos como desenvolver uma página clean e visualmente bonita em poucos minutos, integrando o componente da biblioteca Ext JS com JSON em um projeto Spring, atentando-se para os problemas que podem surgir e mostrando os extras que podem enriquecer ainda mais a página, e é claro, facilitando ao máximo a vida do desenvolvedor.

por Loiane Groner - IBM

Loiane Groner é bacharel em Ciência da Computação pela FAESA, trabalha como Java Developer/IT Specialist num projeto internacional na IBM – Hortolândia-SP, e é instrutora java. Trabalha com Java há 4 anos e trabalhou por 2 anos como Teacher Assistant. É embaixadora técnica e de relacionamento do IBM Academic Initiative no ES. Também é coordenadora do ESJUG e fundadora e coordenadora do CampinasJUG (Grupo de Usuários Java de Campinas-SP e Região).

15:30h - Parada para o Coffee Break

16:00h - Keynote patrocinador: PowerLogic

16:30h - Persistência Java: programando com SQL, Hibernate e BDs não-relacionais

Nessa palestra prática, com código feito na hora, veremos um pouco do que tem acontecido no campo de persistência Java. Desde os primórdios do JDBC com DAO, até o uso do Hibernate com JPA e geração de complexas queries SQL. Passaremos também para o uso dos banco de dados não relacionais (NoSQL), ganhando muita escalabilidade através do modelo de documentos e replicação no cloud.

Microsystems. É atualmente o professor de Java dos cursos de verão da USP. Um dos fundadores do GUJ.com.br e editor técnico da revista Mundo Java. É o responsável pelos cursos da Caelum.
