Ext JS 4 reorganizou os pacote da API, e consequentemente, algumas classes também tiveram mudanças no nome. As classes do Ext JS 3 ainda são válidas no Ext JS 4, mas quando estiver migrando uma aplicação do Ext JS 3 para Ext JS 4, é recomendado trocar o nome das classes também.

Pensando em facilitar essa busca por classes correspondentes do Ext JS 3 no Ext JS 4 criei uma aplicação que lista o nomes das classes que sofreram alterações. Também é possível fazer busca por uma determinada classe e assim encontrar rapidamente a classe correspondente na outra versão do Ext JS. Essa aplicação apenas lista os nomes das classes que mudaram de nome.

Espero que ajude alguém.

Também estou aberta a sugestões para melhorar essa pequena app.

Link: http://loianegroner.com/extjs/examples/ext4-ext3-class-names/
